# AIM1000 – AI Workflow Automation System [https://aim1000-jagan28.streamlit.app/]

## Overview

AIM1000 is a workflow automation and productivity management platform developed using Python, SQLite, Streamlit, Pandas, and Matplotlib.

The system helps users create, manage, track, search, update, and analyze tasks through an interactive dashboard. It is designed to improve productivity by organizing tasks efficiently and providing insights into task completion progress.

## Key Functionalities

### Task Creation

Users can create tasks by providing a title, description, category, priority level, and status.

### Task Management

The application allows users to view, search, update, and delete tasks efficiently.

### Dashboard Analytics

The dashboard displays task statistics including total tasks, pending tasks, completed tasks, and productivity metrics.

### Data Visualization

Task categories are visualized using graphical charts for better analysis and decision-making.

### CSV Export

Users can export all task records into CSV format for reporting and further analysis.

### AI Categorization

The system automatically assigns a category based on task keywords, improving task organization.
